## 📄 Página Inicial — Julia Martins

Desenvolvimento da estrutura base do projeto e da página inicial. Foram criados
os arquivos globais compartilhados entre todas as telas: `styles.css` com o
sistema de design e componentes reutilizáveis, `icons.svg` com os ícones do
projeto, e `app.js` com a lógica da navbar e utilitários compartilhados.

A página inicial conta com navbar fixa e responsiva com menu hambúrguer, seção
hero com imagem de fundo, seção de destaques, cards das pizzas mais pedidas com
fotos reais, banner de chamada para ação e footer completo.
